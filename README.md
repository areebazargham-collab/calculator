Calculator App
A simple and responsive Calculator App built using HTML, CSS, and Vanilla JavaScript.
This project performs basic arithmetic operations with a clean and modern user interface.

Features
Addition, Subtraction, Multiplication, Division
Responsive layout
Interactive button effects
Clear display functionality
Beginner-friendly JavaScript logic
Clean UI design
Technologies Used
HTML5
CSS3
JavaScript

How to Run
Download or clone the repository
Open the project folder
Run index.html in your browser
