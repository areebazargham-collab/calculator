# Calculator App 🧮

A simple and responsive Calculator App built using **HTML, CSS, and Vanilla JavaScript**.  
This project performs basic arithmetic operations with a clean, modern, and user-friendly interface.

---

## ✨ Features

- Addition, Subtraction, Multiplication, and Division  
- Responsive layout for all devices  
- Interactive button effects  
- Clear display functionality  
- Beginner-friendly JavaScript logic  
- Clean and modern UI design  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

---

## ▶️ How to Run

1. Download or clone the repository  
2. Open the project folder  
3. Run `index.html` in your browser  
4. Start using the calculator 🎉  

---

## 📌 Project Purpose

This project is built for practice and learning purposes, especially for beginners who want to understand DOM manipulation and basic JavaScript logic.
